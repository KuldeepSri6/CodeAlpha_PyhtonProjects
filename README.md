### 🎯 2. Hangman Game
A console-based Hangman game where the player guesses a hidden word one letter at a time.

**Features**
- Random word selection
- Limited incorrect attempts
- Displays guessed letters
- Win/Lose detection

**Concepts Used**
- Random Module
- Strings
- Lists
- Loops
- Conditional Statements

---

### 🎯 3. Stock Portfolio Tracker
A Python application that calculates the total investment value based on predefined stock prices.

**Features**
- Accepts stock symbols and quantity
- Calculates investment value
- Displays investment summary
- Saves results to a text file

**Concepts Used**
- Dictionaries
- File Handling
- User Input
- Arithmetic Operations
